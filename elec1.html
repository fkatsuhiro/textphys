導体球殻問題

今回は大学院試験で頻出である導体球殻問題に関して問題を掲載します。

半径 a の導体球と，内半径 b,外半径 c (a < b < c) の中心を同じくする導体 球殻が真空中にある。内球を接地し，外球殻に正の電荷 Q を与えると，内球に電荷 Q" が誘起された。真空中での誘電率を ε0,中心からの距離を r とするとき，以下の問いに 答えなさい。ただし，電場の向きは中心から広がる向きを正とし，無限遠点での電位を基準とする。



① r > c での電場を求めなさい。

② a < r < b での電場を求めなさい。

③ r > c での電位を求めなさい。

④ a < r < b での電位を求めなさい。



解説を見る

【問題を解く際のヒント】

接地時の電位

接地した時の電位に関しては、基本的に電位を0に取ります。また、電位は基本的に滑らかに接続します。 グラフを書けという問題を解く際には、こちらを意識して書くといいです。



導体中の電場

導体中の電場は0です。この結果を用いてガウスの法則から導体球殻の内側表面に励起する電荷などに 関して求めることができます。
